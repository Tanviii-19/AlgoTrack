**AlgoTrack**

AlgoTrack is a clean, interactive DSA Revision Tracker built to help you stay consistent with your coding interview preparation. Instead of keeping notes scattered across notebooks or spreadsheets, AlgoTrack lets you track solved problems, save last-minute revision notes, and monitor your overall progress—all in one place.

## Features

* **Comprehensive DSA Coverage** – Includes all **20 DSA patterns** with **80+ carefully selected interview problems**, organized by topic.
* **Smart Progress Tracking** – Monitor your preparation with real-time category-wise and overall progress bars.
* **Persistent Data Storage** – All progress, notes, favorites, and solution links are automatically saved using **Local Storage**.
* **Last-Minute Revision Notes** – Attach concise revision notes to every problem for quick interview preparation.
* **Powerful Search & Filtering** – Instantly find problems with the built-in search functionality.
* **Favorites & Solution Links** – Mark important problems and save reference links for future review.
* **Category Management** – Mark all problems, reset individual categories, or clear all progress with a single click.
* **Dark & Light Mode** – Toggle between modern dark and light themes for a comfortable coding experience.
* **Responsive Design** – Optimized for desktops, tablets, and mobile devices.
* **Built with Vanilla Web Technologies** – Developed using **HTML, CSS, and JavaScript** with no external frameworks, ensuring fast performance and simplicity.
